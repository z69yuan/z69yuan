Hugo的简单使用

Posted on 2022-05-18 13:22:31 in 工具 • 64 words • 1 minute read
Tags: Hugo使用

一、常用命令

注意：本文当中的命令，都是在 hugo生成的site文件目录，进行执行的。

  1. 创建一个文档

    hugo new post/xxxx.md  //执行该命令，会在 ${site_path}/content/post下生成对应的markdown文件
hugo new post/tools/xxxx.md  //执行该命令，会在 ${site_path}/content/post/tools/下生成对应的markdown文件

    并且生成的markdown文件中，默认有如下：

    title: "Hugo的简单使用"
date: 2022-05-18T13:22:31+08:00
draft: true

    当然，我们可以增加其他的描述信息，如：标签、分类、主题等信息

    tags: ["hugo"]
series: ["Blog养成记"]
categories: ["杂技浅尝"]

  2. 开启Hugo服务，进行本地调试

    hugo server -D  [-p <port_number>]

  3. 部署Hugo服务

    3.1 在"${site_path}/pubic"下生成要部署的静态网页

    hugo server -D # 或者简写  `hugo -D`

    3.2 将public目录下生成的文件，发布到 个人博客（如gitee pages服务）即可。
